Cyber Eyepatch I

2020-05-30

Yo ho, yo ho, a pirate’s life for me.

Unfortunately my right eye does not work all that well, normally it does not look all that different than my left, but it rarely properly lines up with it to allow me to see 3d objects like most people. One project that has been on my to do list for a while now has been creating a cool, cyborg esque eye-patch for that eye. After all, if my left eye is to be basically useless for 3d vision at the least I can construct a cool eye-patch like device to cover it it and give it new senses.

So I have been looking into the requirements for it and developing a plan for the project. Someone else has done it already in 2012 with ultrasonic sensors [which is pretty cool][https://blog.ocad.ca/wordpress/gdes3b16-fw201202-01/2012/12/augmented-vision/] so I am hoping to see if it is possible to improve upon the design. Possible areas of improvement would be the use of LIDAR instead of an ultrasonic distance sensor, the use of a binary sensor to provide allow for the quantification of the incoming data, and the inclusion of other senses with data input separated by the colors of the LEDs. Other possible improvements include the replacement of the Arduino with a Raspberry Pi Zero, allowing me to run linux on my eyepatch.

Though, ultimately, the biggest question is what I want the eye to do as that will decide the capabilities I will need for it. Generally I want it to measure distance so that I may be able to accurately estimate the speed of oncoming objects and so on. I would also like to make it modular, so that I can add new tools to the eye and perhaps link it to my computers over bluetooth or a USB to monitor different processes and so on.
